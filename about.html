What is Interview Armor: Dev Mode?

A space to say your answers out loud.

Interview Armor: Dev Mode is a web developer interview prep app built around speaking, not just reading. Instead of scrolling through question lists or typing rehearsed answers, you record yourself, play it back, get a clean transcript, and run it through AI feedback.

The goal is to build the muscle for thinking and speaking on your feet — before the real interview lands.

Built For

Devs walking into the room ready.

  • Coding bootcamp students
  • Career changers transitioning into tech
  • Web developer applicants
  • Anyone prepping for a technical interview
  • People who want to practice speaking with confidence

Built With

The stack under the hood.

HTML CSS Vanilla JavaScript SPA Router MediaRecorder API Groq Whisper Groq Llama 3.3 localStorage

Current Version

What's working today.

  • Single-page navigation — no full reloads between tabs
  • developer interview question bank with level + category filters
  • Mock interview recording with audio playback
  • Speech-to-text transcription via Groq Whisper
  • AI coaching feedback with clarity, structure, and confidence scores
  • Saved transcripts and feedback for review
  • Auto-saved drafts so you can pick up where you left off
