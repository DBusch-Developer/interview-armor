About Interview Armor

------------------------------------------------------------

[ ABOUT THE APP ]

 --------------------------------------------------------
| What is Interview Armor: Dev Mode?                    |
|                                                        |
| Interview Armor is a junior web developer interview    |
| prep app that helps users practice speaking their      |
| answers out loud.                                      |
|                                                        |
| Instead of only reading questions or typing answers,    |
| users can record themselves, review playback, get a    |
| transcript, and receive AI-powered feedback.           |
 --------------------------------------------------------

------------------------------------------------------------

[ WHO IT IS FOR ]

 --------------------------------------------------------
| Built For                                              |
|                                                        |
| - Coding bootcamp students                             |
| - Career changers                                      |
| - Junior web developer applicants                      |
| - People preparing for technical interviews            |
| - People who want to practice speaking with confidence |
 --------------------------------------------------------
